# Medicare Report Generator - Quick Usage Guide

## Test the Tool

### Option 1: Run Test Script (Recommended)

```bash
cd backend
npx tsx src/test-medicare-report.ts
```

This will generate a sample Medicare report PDF in `backend/reports/`.

### Option 2: Use via Chat Interface

Open the frontend at `http://localhost:3000` and try these prompts:

**Example 1: Simple Request**
```
Generate a Medicare report for a consultation with John Smith on 2024-11-24. 
He had a cough and fever. Use MBS code 23 ($91.80).
```

**Example 2: Detailed Request**
```
Create a Medicare report with the following details:

Patient: Jane Doe, DOB: 1985-06-20, Medicare: 3456 78901 2
Provider: Dr. Michael Chen, Provider Number: 234567CD
Consultation: 2024-11-24, Standard Consultation, 30 minutes
Notes: Patient presented with headache and dizziness for 2 days. 
       Examination normal. Advised rest and hydration.
MBS Codes: 
- Item 36 ($150.00) - Extended consultation
- Item 721 ($25.00) - Health assessment
```

The AI agent will automatically call the `generate_medicare_report` tool and create a PDF.

## What You'll Get

- **PDF Report** saved in `backend/reports/`
- **Professional Clinical Summary** generated by AI
- **Formatted MBS Codes** with fees and descriptions
- **Total Fee Calculation**
- **Medicare-compliant structure**
